Hexo使用方法

Hexo的详细使用文档详见这里。

Hexo是一个博客框架，可以将本地的markdown文档渲染成静态网页并布署到github。

Hexo的安装

  • Node.js环境 (因为Hexo使用Node.js编写，所以必须安装这个环境)

  • Git环境 (因为需要与Github互相沟通，所以必须安装)

    具体见此处说明。

  • Hexo本身的安装

    $ npm install -g hexo-cli

    npm是Node package manager, -g是全局安装

    此句的意思是使用npm全局安装hexo-cli这个软件。
