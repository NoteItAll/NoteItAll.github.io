HTML语言

HTML中文全名叫做：“超文本标记语言”。超文本的意思就是不止是文本，还可以包含图片，链接，音乐，甚至程序等非文字元素。web developer使用html语言告诉浏览器如何生成网页。

flowchart LR
    A[HTML<br>CSS<br>javascript<br>...] --> B(浏览器)
    B --> C[网页]

A Simple HTML Document

HTML语言不区分大小写，但推荐全使用小写。

<!-- 声明文档类型为html5,所有html文档必须有此声明，必须在最开始且只能声明一次-->
<!DOCTYPE html>					
<!-- <html>作为整个html文件的起始-->
<html lang="en-US">							
    <!--  <head>作为meta information元信息的起始 lang是声明网页语言-->
    <head>						
         <!--  显示在浏览器标签上的标题-->
        <title>Page Title</title>		
      </head>
    
    <!-- <body>作为网页主体的起始-->
    <body>							
        <!--  一级标题，按重要性一共分6级，分别为h1到h6-->
        <h1>My First Heading</h1>	
        <!-- break打断，新起一行-->
        <br>						
        <!-- paragraph段落，也即正文-->
        <p>My first paragraph.</p>	
        <!-- 使用红色-->
        <p style="color:red;">This is a red paragraph.</p>
        <!-- a: Anker（锚）-->
        <!-- href: hypertext reference(超文本引用)-->
        <a href="https://www.w3schools.com">This is a link</a>
           <!-- 图片，src为图片的路径，alt为替代文字-->
        <img src="w3schools.jpg" alt="W3Schools.com替代文字" width="104" height="142">
    
    </body>
</html>

查看网页的源代码

查看HTML源代码：

在一个HTML页面上点击右键，选择 “查看页面源代码”（在Chrome中）或 “查看源代码”（在Edge中），或在其他浏览器中选择类似的选项。这将打开一个包含该页面HTML源代码的窗口。

检查一个HTML元素：

在一个元素（或一个空白区域）上点击右键，选择 “检查 “或 “检查元素”，就可以看到哪些元素是由什么组成的（你会同时看到HTML和CSS）。你也可以在打开的元素或样式面板中即时编辑HTML或CSS。

HTML Attributes

HTML 元素可以有多个参数，参数应当包含在元素的起始标签中，格式为**name=”value”**。

<p title="当鼠标悬停时显示我">This is a paragraph.</p>

value两侧加单引号或双引号，但有时候当想显示双引号时，必须使用单引号外括，反之亦然。

<p title='John "ShotGun" Nelson'>鼠标悬停时显示双引号</p>
<p title="John 'ShotGun' Nelson">鼠标悬停时显示单引号</p>

完整的HTML参数见：HTML Attribute Reference。

标题

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

标题并非只是加大加粗而已，这是一种专门类别的文字，可以帮助搜索引擎索引到你的页面上来。

各种标题有默认的大小，当然也可以自定义size

<h1 style="font-size:60px;">Heading 1</h1>

正文

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

<hr>可以加一条水平分割线 (hr: horizontal line)

<br>可以加一个分割，形成新行 (br: break)

<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<br>
<h2>This is heading 2</h2>
<p>This is some other text.</p>

浏览器会自动忽略正文中的空格、回车等标记，下面的诗会显示到一行

<p>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</p>

为解决这个问题，可以使用pre标签

<pre>可以保存段落的空格和分行号 (pre: preserve)

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>

HTML Styles

格式为

<tagname style="property:value;">

下面命令可以使段落变红、蓝、尺寸变大

<p>I am normal</p>
<p style="color:red;">I am red</p>
<p style="color:blue;">I am blue</p>
<p style="font-size:50px;">I am big</p>

将正文的背景换色(默认白色)

<body style="background-color:powderblue;">

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>

将单独的元素背景换色

<body>

<h1 style="background-color:powderblue;">This is a heading</h1>
<p style="background-color:tomato;">This is a paragraph.</p>

</body>

改变元素的颜色

<h1 style="color:blue;">This is a heading</h1>
<p style="color:red;">This is a paragraph.</p>

改变字体

<h1 style="font-family:verdana;">This is a heading</h1>
<p style="font-family:courier;">This is a paragraph.</p>

改变字的大小

<h1 style="font-size:300%;">This is a heading</h1>
<p style="font-size:160%;">This is a paragraph.</p>

改变字的对齐方式

<h1 style="text-align:center;">Centered Heading</h1>
<p style="text-align:center;">Centered paragraph.</p>

文本格式

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <mark> - Marked text
  • <small> - Smaller text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text

可以将文本变小

<small>This is some smaller text.</small>

删除线

<p>My favorite color is <del>blue</del> red.</p>

插入文字，通常下面加一条下划线

<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>

文字下标、上标

<p>This is <sub>subscripted</sub> text.</p>
<p>This is <sup>superscripted</sup> text.</p>

Quotation and Citation Elements

  • <blockquote> - 插入一段引用，自动缩进

  • <q> - 插入一句话，两侧自动添加上引号

  • <abbr> - 缩写，自动加虚下划线

  • <address> - 地址，自动斜体

  • <cite> - 引用，自动斜体

  • <bdo> - bi-directional override

<p>WWF's goal is to: <q>Build a future where people live in harmony with nature.</q></p>

缩写，自动，在下面加上虚下划线。title是鼠标悬停时显示的内容

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>

bdo需要指定方向。dir只有两种方向，rtl即right to left，ltr即left to right。

<bdo dir="rtl">This text will be written from right to left</bdo>

本节内容其实不太常用，具体可见这里。

Comments

Colors

加背景色

<h1 style="background-color:DodgerBlue;">Hello World</h1>
<p style="background-color:Tomato;">Lorem ipsum...</p>

加文字颜色

<h1 style="color:Tomato;">Hello World</h1>
<p style="color:DodgerBlue;">Lorem ipsum...</p>

加边框颜色

<h1 style="border:2px solid Tomato;">Hello World</h1>
<h1 style="border:2px solid DodgerBlue;">Hello World</h1>

颜色也可以用RGB值、HEX值、HSL值、RGBA值和HSLA值来指定。

<!-- 下面这三种都是橘红色 -->
<h1 style="background-color:rgb(255, 99, 71);">...</h1>
<h1 style="background-color:#ff6347;">...</h1>
<h1 style="background-color:hsl(9, 100%, 64%);">...</h1>

<!-- alpha代表透明度 transparency -->
<h1 style="background-color:rgba(255, 99, 71, 0.5);">...</h1>
<h1 style="background-color:hsla(9, 100%, 64%, 0.5);">...</h1>

RGB and RGBA

rgb(red, green, blue)

rgba(red, green, blue, alpha)

alpha=0时完全透明，alpha=1时完全不透明

HEX

格式#RRGGBB, 其中 RR (red)，GG (green) ，BB (blue)

比如黑色#000000，白色#ffffff

HSL and HSLA

hsl(hue, saturation, lightness)

hsla(hue, saturation, lightness, alpha)

Hue 色相是色轮上从0到360的一个度数。0是红色，120是绿色，而240是蓝色。

Saturation 饱和度是一个百分比值。0%意味着灰色的阴影，而100%是完整的颜色。

Lightness 明度也是一个百分比值。0%是黑色，100%是白色。

style - CSS

CSS 即 Cascading Style Sheets （层级样式表）层级的意思是，如果你在父元素施加了红色的文字颜色，那么子元素也会自动有红色的文字颜色，除非再另行指定。

CSS 有三种施加方式

  • Inline - 在元素内加style参数
  • Internal - 在<head>段内加<style>元素
  • External - 使用<link>元素链到一个外部CSS文件。

Inline 的例子

<h1 style="color:blue;">A Blue Heading</h1>

<p style="color:red;">A red paragraph.</p>

Internal 的例子

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color: powderblue;}
h1   {color: blue;}
p    {color: red;}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

External 的例子

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

styles.css

body {
  background-color: powderblue;
}
h1 {
  color: blue;
}
p {
  color: red;
}

可见，如果想修改多个网页的格式，只需要修改这个css文件即可，这是很方便的。

CSS Colors, Fonts and Sizes

color 颜色

font-family 字体

font-size 字体大小

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
  color: blue;
  font-family: verdana;
  font-size: 300%;
}
p {
  color: red;
  font-family: courier;
  font-size: 160%;
  border: 2px solid powderblue
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

border 边框

padding 边框内距

margin 边框外距

p {
  border: 2px solid powderblue;
  padding: 30px;
  margin: 50px;
}

链接到外部CSS文件，分别是url地址，相对路径，当前文件夹。

<link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">
<link rel="stylesheet" href="/html/styles.css">
<link rel="stylesheet" href="styles.css">

links

语法

<a href="url">link text</a>

例子

<a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>

target 参数

  • _self - 在自身窗口打开
  • _blank - 在一个空窗口打开
  • _parent - 仅父框架跳转
  • _top - 整个窗口一起跳转

使用图片做超链接

<a href="default.asp">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>

使用邮件做超链接

<a href="mailto:someone@example.com">Send email</a>

使用按钮做超链接

<button onclick="document.location='default.asp'">HTML Tutorial</button>

title - 鼠标悬停时显示的内容

<a href="https://www.w3schools.com/html/" title="Go to W3Schools HTML section">Visit our HTML Tutorial</a>

可以对链接的各种情况进行定制

<style>
a:link {
  color: green;
  background-color: transparent;
  text-decoration: none;
}

a:visited {
  color: pink;
  background-color: transparent;
  text-decoration: none;
}

a:hover {
  color: red;
  background-color: transparent;
  text-decoration: underline;
}

a:active {
  color: yellow;
  background-color: transparent;
  text-decoration: underline;
}
</style>

Bookmarks

创建一个书签，并建立一个链接跳至这个书签。

<h2 id="C4">Chapter 4</h2>
<a href="#C4">Jump to Chapter 4</a>
<a href="html_demo.html#C4">Jump to Chapter 4</a>

图片image

<img src="pic_trulli.jpg" alt="Italian Trulli">

alt 中为图片无法显示时呈现的内容

指定其宽和高

<img src="img_girl.jpg" alt="Girl in a jacket" style="width:500px;height:600px;">

<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

也可在<head>中指定全部 img 的格式

<style>
/* This style sets the width of all images to 100%: */
img {
  width: 100%;
}
</style>

使用CSS float 属性使图片处在段落的右边或左边

<p><img src="smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px;">
The image will float to the right of the text.</p>

<p><img src="smiley.gif" alt="Smiley face" style="float:left;width:42px;height:42px;">
The image will float to the left of the text.</p>
