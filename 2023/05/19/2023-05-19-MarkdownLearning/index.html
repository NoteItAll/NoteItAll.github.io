Markdown 标记语言

Markdown标记语言使人们可以摆脱杂乱的html文本，对于简单的个人博客类静态网页来说非常适用，相应的编辑软件Typora更是把这种极简风格发挥得淋漓尽至，在书写过程中甚至不太能感到自己在编辑网页。

Markdown的基本语法也是很简单易学的。这里是Markdown的官方教程文档：https://markdown.com.cn/basic-syntax/

1-6号标题分别为在文字前面添加1到6个 # 号，文字前要留一个英文空格。

有序列表

直接以1. 开始一句话

1. 我先做饭
2. 我再吃饭
    1. 我先吃肉
    2. 我不吃菜
3. 我后洗碗

  1. 我先做饭

  2. 我再吃饭

    1. 我先吃肉
    2. 我不吃菜

  3. 我后洗碗

无序列表

以 * 或 + 或 - 引导文本

* 我先做饭
+ 我再吃饭
    - 我先吃肉
    + 我不吃菜
+ 我后洗碗
  • 我先做饭
  • 我再吃饭
    • 我先吃肉
    • 我不吃菜
  • 我后洗碗

不同的 Markdown 应用程序处理URL中间的空格方式不一样。为了兼容性，请尽量使用%20代替空格。
